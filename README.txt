
# Maine Hindu Temple — Static Site

This bundle includes a clean, mobile‑friendly multi‑page site for GitHub Pages.

## How to use
1. Download the ZIP and unzip it.
2. In your repo `mainehindutemple2025/mainehindutemple`, upload/replace files in the root (keep `mht.jpg`).
3. Commit to `main`. GitHub Pages will deploy automatically.
4. Update placeholders:
   - Address, phone, email (in `contact.html` and footer in all pages).
   - Timings and donation amounts.
   - Events list in `events.html`.
   - Connect forms (Formspree/Getform/Netlify) or remove `action="#"`.

## Optional
- Add a payments link (PayPal/Stripe/Zeffy) to buttons in `donate.html`.
- Replace placeholder images with your own.
- Edit theme colors in `assets/styles.css`.
